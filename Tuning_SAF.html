Source and Analyzer Focus

A Source and Analyzer Focus (SAF) adjusts the flight tube voltages in order to optimize the intensity, resolution, and symmetry of the spectral peaks produced by the instrument. This is an automatic tune; no user intervention is required. The following screenshot is an example of what the user might see on the screen during a SAF. Note, this window can be minimized, but cannot be closed while the system is activley acquiring an SAF.

Source and Analyzer Focus Stage Tracking Viewer.

In general, users can ignore the information that is displayed here, however here is a brief description of the major components:

  • The Stage tabs can be used as a way to monitor how many stages are complete and how many are left.
  • The Details button shown on each tab display advanced information about the status of the tune.
  • The Track active stage check box will cause the tab for each stage to be automatically activated when its stage begins.
  • Export data allows the user to export the raw data collected during the tune in CSV format.

A SAF can be added to the Acquisition Queue either as part of an Auto-Tune or individually as a single line. There are two videos below, the first demonstrates how to add a SAF to the Acquisition Queue individually, and the second shows what a SAF looks like as it is being acquired.
