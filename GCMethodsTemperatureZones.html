Gas Chromatograph Methods: Temperature Zones

Inlet Temperature

  • Different areas of Inlet Temperature are active and visible depending on the type selected for the front and/or back inlet in the Inlet Configuration. When None is selected as the Inlet Type, the Inlet Temperature section will not display.

Oven Tracking

  • When On-Column is selected, the Oven Tracking checkbox is available. Select the track Oven Temperature checkbox and the inlet temperature automatically stays 3 °C higher than the oven temperature throughout the run. Select the Do Not Track Oven Temperature checkbox, and the inlet temperature is controlled by the parameters entered in the temperature program table.

Inlet Temperature Table

  • When Split/Splitless, PTV, On-Column-Do Not Track Oven Temperature, or Gerstel are selected for Inlet Type, the Inlet Temperature Table is available.
  • For Initial Inlet Temperature, enter the beginning temperature for the inlet.
  • For Time at Initial Inlet Temperature, enter the length of time the inlet stays at initial inlet temperature before beginning the programmed temperature changes. If the entire analysis consists of one temperature, enter the desired temperature in the Target Temp column and the length of the analysis in the Duration column. If the column temperature should ramp:
    1. Select Add to add rows for the desired number of temperature changes (levels), or select Remove to remove the desired number of rows.
    2. In the Rate column, enter the rate of temperature change for each temperature level.
    3. In the Target Temp column, enter the final temperature for each temperature level.
    4. In the Duration column, enter the length of time for each temperature level.

Cryogenic Cooling

  • When PTV or On-Column are selected as Inlet Types, the Cryogenic Cooling checkboxes will be available.
  • For Inlet Cryo, select On to activate cryogenic cooling to cool the inlet. Select Off, and cryogenic cooling will not be activated.
  • For Cryo Timeout, select On to activate cryogenic timeout; select Off, and Cryogenic Timeout will not be activated. When a run does not start within the specified time, Cryo Timeout begins and inlet temperature shuts down.
  • For Cryo Fault, select On to activate cryogenic fault; select Off, and Cryogenic Fault will not be activated. Cryo Fault shuts down the inlet temperature if it does not reach set point in 16 minutes of continuous Cryo operation.

Oven Temperature

  • When the oven temperature is valid, a green icon will display with a message that displays “No problems detected with oven temperature.” When the oven temperature is not valid, a red icon will display with an error message that states the cause of the problem.
  • Different areas of Oven Temperature are available depending on the conditions entered.
    1. Enter the oven equilibration time. This value specifies the time allowed for the oven temperature to equilibrate after the temperature is modified. Equilibration time begins when the actual oven temperature comes within 1 °C of the oven temperature setting.
    2. Complete the Oven Temperature Ramp table.
      • Select Add to add rows for the desired number of temperature changes (levels).
      • In the Rate column, enter the rate of temperature change for each temperature level.
      • In the Target Temp column, enter the final temperature for each temperature level.
      • In the Duration column, enter the length of time for each temperature level.
      • Or
      • To remove rows, select the row and then select Remove.
    3. For Coolant to Column Oven, select either On or Off. When On is selected, coolant will be used to cool the oven, and the box for the Coolant Timeout is available. Enter maximum time for the oven to reach set point. If the set time is exceeded the coolant will be shut off.
    4. Enter the transfer line temperature equilibration time. This specifies the length of the wait to ensure the transfer line is at temperature. Time set to zero is the standard mode of operation.
    5. Enter the transfer line temperature in °C. Generally choose a temperature value of 10 °C above the maximum oven temperature for the Gas Chromatograph, as long as this temperature value is compatible with the column stationary phase. If the temperature displays as –1 because the method was created using a previous version of the software, update the temperature.

GCxGC Specific Parameters

Programming Secondary Oven

  • Add a capillary column and change the location to Secondary Oven on the Columns page to make the Secondary Oven Temperature Offset available. Define the temperature profile for the secondary oven over the length of the run, which will be relative to the GC oven temperature.
  • Or
  • Select the checkbox for Use Advanced Secondary Temperature Programming to manually program the temperature settings if custom temperature programming is desired.
  • For a temperature program:
    1. Select Add to add a step at a particular heating rate. The starting temperature must be at least 5 degrees above the starting temperature of the GC oven because the secondary oven cannot cool as low as the primary oven due to heat from the transfer line and modulator. Or, to remove a row, select the row and select Remove.
    2. In the Rate column, enter the rate of temperature change for that step.
    3. In the Target Temp column, enter the final temperature for that step.
    4. In the Duration column, enter the length of time for that step. The initial Duration has to be at least 0.2 minutes. This is necessary for temperature control due the thermal lag of the secondary oven.

Modulator Temperature Programming

  • Add a capillary column and change the location to Modulator on the Columns page to make the Modulator Temperature Offset available. Define the temperature profile for the modulator over the length of the run, which will be relative to the secondary oven temperature. If a secondary oven is not being used, the modulator temperature offset is relative to the GC oven. The recommended offset is +15 °C relative to the temperature of the secondary oven. A smaller difference may increase the widths of the first dimension peaks. The modulator will heat at the same rate as the primary oven up to the maximum temperature defined for the modulator on the Columns page.
  • Or
  • Select the checkbox for Use Advanced Modulator Temperature Programming to manually program the temperature settings if custom temperature programming is desired. Follow the same instructions above for the programming of the secondary oven.
