Data Processing: Status Monitoring

Introduction

The processing queue and monitor is an accessory to the data processor within ChromaTOF-BT. Together these two windows allow the user to monitor the status of a data processing job, stop a data processing job, or reorder the sequence in a list of queued jobs.

Using the Data Processing Queue & Monitor

The video shows where the Data Processing Queue & Monitor view are located within the ribbon. Furthermore, it demonstrates how to promote a particular data processing job up or down the sequence of DP jobs. Lastly, it illustrates how to remove a series data processing jobs from the queue.
