Auto-Tune

Auto-tune automatically runs the recommended set of tunes for the Pegasus BT. This feature streamlines the process of tuning so that the user does not have to manually add each of the tuning tasks. Despite the fact there is a single line in the Acquisition Queue each of the recommended tuning tasks are carried out and a corresponding file for each is created into the defined Acquired Sample directory. For all versions of ChromaTOF up to 5.31.16 this includes the following:

  • Source and Analyzer Focus
  • Mass Calibration
  • Gain Optimization
  • Tune Validation

The video below demonstrates how to implement an Auto-Tune using the acquisition queue. Notice that Gas Chromatographic and Mass Spectrometer methods are included. This is done to ensure source conditions are constant from tuning and sample acquisition.
