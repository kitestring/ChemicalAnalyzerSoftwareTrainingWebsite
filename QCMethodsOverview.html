Quality Control Methods

Introduction

The Quality Control (QC) method controls the setup for automatically scheduling instrument optimization, detector drift, instrument checks, and sample runs.

Method Description

  • The Events tab for the QC Method lists events and what actions they will automatically trigger if the QC Method is specified in the Acquisition Queue.
  • For each section on the Event page, the user can select the desired course of action by either selecting from the drop-down list or setting the checkbox(es) to selected (will run) or cleared (will not run). Even though the settings can be changed, using the default Events settings is highly recommended.
  • On the other pages of the method, the user can specify when the tune should be scheduled (based on hours and/or the number samples run) and what Acquisition Queue settings should be used when the tune actually runs.
  • For the Selected Parameters tabs, select the checkbox for hours and/or samples.
    • Select hours to specify the maximum amount of time that has passed since the last check was performed. If more than the specified time has elapsed, the check is automatically scheduled prior to running the next sample.
    • Select samples to specify the maximum number of samples that have been collected since the last check was performed. Once the maximum number of samples has been reached, the check is automatically scheduled prior to running the next sample.
    • Complete the Acquire table by selecting the desired folder and name, and optionally specify methods to use when the tune runs. These values will be used in the Acquisition Queue.
  • Leak Check has the following additional test parameters:
    • Select the checkbox next to Open Tune Compound Valve to allow the tune compound to enter the system during the leak check.
    • Fill out the Criteria table with the criteria that must be satisfied in order for the leak check to pass. The leak check is for the vacuum chamber of the mass spectrometer.
  • Both the carrier gas used and the state of the tune compound valve checkbox (On or Off) should be taken into account when setting up criteria for the leak check.
