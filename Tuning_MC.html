Mass Calibration

Introduction

A mass calibration tune derives an equation for converting between TOF and m/z. Using a second order fit m/z values from known ions are correlated to their measured TOF (ns) to determine the coefficients C2, C1, and C0. The known ions are derived from the calibration gas, Perfluorotributylamine (PFTBA), and reproducible background ions. See the plot below for a representative example of a Mass Calibration plot.

Acquiring a Mass Calibration

Mass calibration can be executed during system optimization or after sample acquisition using a Data Processing method. To mass calibrate during system optimization select Mass Calibration in the Type field of the Acquisition Queue, and define a Mass Spectrometer and Gas Chromatograph method to be used. Reviewing the Mass Calibration

There are two options available to review the Mass Calibration results.

  1. The Tune Validation report will show the mass accuracy root mean square (RMS).
  2. The Mass Calibration Table will list each of the ions used to generate the Mass Calibration curve as well the associated metrics. Additionally, the three Mass Calibration coefficients for the second order polynomial fit for the calibration curve as well as the RMS can be viewed via the Mass Calibration Table right click menu. Reviewing the Mass Calibration results via the Mass Calibration table.
  • Mass Calibration Table Fields
    • Name: This is a text field intended for easy identification of a particular ion. The name itself has no impact on the mass calibration calculations.
    • Priority: Typically the mass calibration curve is not calculated in one step. Setting different priority levels allows ChromaTOF to create a rough curve based on the most prevalent masses. Then, using this temporary curve ChromaTOF will seek out and find the smaller, secondary masses. These steps will be repeated for as many priority levels as are included in the mass calibration table with priority 1 masses going first, followed by priority 2 and continuing through all priorities that have been defined.
    • Mass Window: The mass range around the expected m/z in which ChromaTOF will look for a spectral peak. If more than one peak exists in this window then ChromaTOF will choose the largest one to use for mass calibration. For priority 1 fragments the windows are typically set relatively wide to ensure the ions are within the expected windows. After a preliminary mass calibration equation is generated from the priority 1 ions, further priorities mass windows can be located with better accuracy. Therefore tighter mass windows can be used with subsequent priorities.
    • Start/End Time: The 1st dimension time range to gather the spectral data for the mass calibration.
    • TOF: The time of flight measured in nano-seconds a particulr ion required to leave the source and reach the detector.
