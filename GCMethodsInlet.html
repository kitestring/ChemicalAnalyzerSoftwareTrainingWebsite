Gas Chromatograph Methods: Inlet

Inlet Configuration

  1. Select the Front Inlet Type and/or Back Inlet type depending on the capillary configuration by selecting the drop-down arrow and making a selection from the list.

    Inlet Types:

    • Select Split/Splitless, PTV, or On-Column depending on requirements. Refer to the Agilent User Information CD for further information about these inlets.
    • Select Gerstel when the Gerstel CIS4 inlet is installed. When this inlet is selected, the inlet temperature parameters are not downloaded to the GC. The Gerstel external controller controls the inlet temperature.
    • Select None when a non-supported inlet or no inlet is installed.
  2. Select the Front Inlet Mode and/or Back Inlet Mode depending on the capillary configuration by choosing a selection from the list. Inlet Mode is not active if either On-Column or None is selected for Inlet Type.

    Inlet Types:

    • Select Splitless for trace concentrations. Splitless operates with the split vent closed.
    • Select Split when only part of the sample goes onto the column. Split operates with the split vent open.
    • Select Pulsed Splitless to increase pressure in the inlet during a splitless injection.
    • Select Pulsed Split to increase pressure in the inlet during a split injection.
    • Select Solvent Vent for a splitless injection with the split vent open to flush solvent from the inlet. Solvent Vent is available with a PTV inlet.
  3. If both columns are defined, select the Active Inlet Location. If only one column is defined, the software automatically selects it, and this option is not displayed.
    • If both columns are defined, there will be separate Front and Back tabs for the Inlet page, where information for the respective inlet can be entered. Temperature affects all plumbed inlets and requires flow in order to avoid damage. If both inlets are plumbed, remember to configure the inactive inlet as well as the active inlet.

Flow Mode

If the following parameter values are not entered correctly, then the column and split flows calculated will be inaccurate. When the pressure/flow configuration is valid, a green icon appears with a message that displays “No problems detected with the pressure/flow.” When the pressure/flow configuration is not valid, a red icon appears with an error message that displays the cause of the problem.

Corrected Constant Flow Via Pressure Ramps Checkbox

Select Corrected Constant Flow when using one very short column or when using a multi-capillary configuration. When the checkbox is selected, enter the desired column flow. The software automatically adjusts all settings. Verify the column configuration is entered correctly. Clear the checkbox to select one of the column modes, which are explained as follows.

Column Modes

  • Select Constant Pressure and then complete the front inlet and/or back inlet pressure table, depending on whether or not column flow path 2 is selected. In the table, enter the pressure for the chromatographic run to maintain the column pressure at the head of the column throughout the run. As the gas viscosity changes, mass flow rate changes, but the gauge pressure does not change.
  • Select Constant Flow and then complete the front inlet flow and/or back inlet flow, depending on whether or not column flow path 2 is selected. In the table, enter the flow rate for the carrier gas through the column to maintain a constant flow rate of the carrier gas in the column throughout the run. As the gas viscosity changes due to a temperature program, the column head pressure is adjusted to keep the flow constant. The flow will only be accurate for a single column >5m and one temperature zone. With the Constant Flow mode, the gas flow through the column is supposed to be constant, but the GC may not correctly calculate a constant flow due to a number of enhanced column configurations supported by ChromaTOF®. Corrected constant flow takes these configurations into account.
  • Select Ramped Flow to vary the flow rate in the column during the run according to the defined flow program. Complete the Flow Rate Program Table to set the parameters that control the flow rate and timing of flow rate in column. For the Initial Flow, enter the beginning flow rate for the carrier gas through the column; for Time at Initial Flow, enter the length of time flow rate stays at initial flow before beginning the programmed flow rate changes.
  • Select Add to insert a ramping step to the table.
  • Select Remove to delete a ramping step from the table.
  • Select Ramped Pressure to vary column head gauge pressure during the run according to the defined program. Complete the Pressure Program Table to set the parameters that control the pressure and timing of pressure in column. The pressure program is limited to three steps.
  • Select Add to insert a ramping step to the table.
  • Select Remove to delete a ramping step from the table.

Constant Flow Correction

The following section provides additional information about the flow modes when a column configuration other than the standard GC column configuration is used.

The four flow modes of Constant Flow, Ramped Flow, Constant Pressure, and Ramped Pressure are based on the flow control capabilities of the Agilent GC. The control of flow by the Agilent GC is based on the dimensions (length and inner diameter) of a single capillary, GC oven temperature, and inlet and outlet pressures. However, depending on the capillary dimensions and the effect of the transfer line on flow, the flow calculated by the GC can have significant error. This error is in accuracy and not precision of flow control. Even with the accuracy error, the flow control of the Agilent GC will still be very good.

For conventional GC, the error in flow is quite small and not significant, which means the Agilent GC flow modes are appropriate in typical single-column situations. In order for the GC to control flow accurately throughout the chromatographic run, all of the following must be true:

  • In the capillary configuration, all capillaries are located in the GC oven.
  • The total length of the capillaries exceeds 5 meters.
  • The GCxGC option is not selected.
  • A flow combiner is not in use.
  • All capillaries are the same ID.

In the case where any of the previous requirements are not met, the GC will only accurately set the flow at the start of a run. In this case it is more appropriate to use Corrected Constant Flow Via Pressure Ramps.

The location of each capillary must be explicitly specified in the capillary configuration.

Single Column Configuration

  • Constant Flow Mode
  • The Agilent GC flow modes are appropriate for single column configurations >5m (for example, only a single column is specified in the capillary configuration table and no transfer line is specified). The ChromaTOF software downloads column dimensions entered, and the GC calculates column flow and split flow based on the column dimensions, oven temperature, and split ratio specified.
  • Corrected Constant Flow Via Pressure Ramps Mode
  • This mode approximates constant flow by linear pressure ramps based on the capillary configuration table. This mode is recommended in general whenever the multiple capillaries are of different ID and/or in different temperature zones. An equivalent column length is downloaded to the GC so that the GC calculates accurate column and split flows at initial conditions. The pressure program then approximates constant flow throughout the temperature program.
  • This mode is not necessary for single column configurations unless a very short column is used (i.e., 5 meters or less). When a very short column is used, the short portion of column in the transfer line becomes a significant part of the restriction at the high transfer line temperature when the oven temperature is low. Using the constant column flow mode of the Agilent GC can result in an initial flow error of >10% when the oven is at 40 °C and the column is 2 m. This error decreases to zero as the oven temperature goes to the temperature of the transfer line. This error also increases as the column becomes shorter. This error is in flow accuracy and not precision. Even with the flow error, precision is very good and will provide good retention time reproducibility.
  • There are conditions under which this mode may be disabled. These conditions include the following though other situations are possible:
    • More than three pressure ramps are required. A maximum of three pressure ramps is possible. To reduce the number of pressure ramps, reduce the number of temperature ramps and/or temperature holds of different lengths for the multiple temperature zones.
    • An incorrect column configuration that will not allow the software to calculate pressures to approximate a flow.
    • When a real, physically meaningful result cannot be calculated. Examples of situations where real, physically meaningful results cannot be calculated would be cases involving the optional flow combiner where certain flows or pressures cannot be achieved because of the other pressures or flows specified. A specific example, though other examples exist, would be when a subambient pressure in one of the inlets would be required.

Multi-Capillary Configurations

  • Configurations with multiple capillaries may use the GC flow modes if all capillaries located in the GC oven are the same internal diameter. However, when more than one capillary is used in a column configuration, the Corrected Constant Flow Mode is typically recommended.
  • The capillary configuration is specified in the Column Configuration Table available from Instrument on the System ribbon bar.
  • If the GCxGC option is selected, the definition in the capillary configuration table must agree with whether the secondary oven checkbox is selected or cleared in the GC Method
  • For all multi-capillary configurations, an equivalent column length is calculated for all flow modes (this is different from a single column) so that the split flow is accurately calculated from an accurate column flow and the entered split ratio. The user will see this equivalent length for the column dimensions displayed on the GC. The equivalent length is based on all of the capillaries entered in the Column Configuration Table and their temperatures.

Actual Flow in Different Modes

  • Constant Flow Mode Flow is accurate at the beginning because of the equivalent length column downloaded. This is different from single column constant flow mode. As the temperature increases, the actual flow increases relative to the target flow. The magnitude of the error in flow will depend on the lengths, inner diameters (ID), temperatures of the columns, and capillary transfer lines.
  • If the multiple capillaries are the same ID, total length > 5 m, and their temperatures are the same, the error will be very small. If one of the columns is short with a smaller ID (typical GCxGC) and/or the temperature zones are very different, the error will be large.
  • Ramped Flow Mode offers the same behavior as constant flow mode, but flow is programmed.
  • Flow with Constant Pressure Mode depends on overall restriction of combined columns and transfer line. An equivalent column is calculated for the initial conditions. This allows the GC to calculate accurate column and split flows at the initial conditions.
  • With Ramped Pressure Mode, an equivalent column is calculated for the initial conditions the same as for the constant pressure mode so that the initial column and split flows are accurate.

Split/Purge

  • The following section explains the different areas of Split/Purge that are active depending on the conditions entered for the Inlet Mode located in Inlet Configuration.
  • If Splitless is selected for the Inlet Mode, enter parameters for column 1 and/or 2 and the front and/or back inlet, depending on the capillary configuration.
    1. For the Inlet Purge Time, enter the time after the beginning of the run when the purge valve opens.
    2. For the Inlet Purge Flow, enter the rate of flow from the purge vent.
    3. For the Inlet Total Flow, enter the actual flow rate to the inlet during a Pre-Run and a run before the purge time.
    4. For the Gas Saver, select Yes or No for Column 1 Front Inlet Gas Saver and/or Column 2 Back Inlet Gas Saver. Select No, and the flow of carrier gas will not be reduced; select Yes to reduce the flow of carrier gas into the inlet and out of the split vent after the completion of the injection. This does not alter the carrier gas flow through the column. When Yes is selected, Gas Saver Flow and Gas Saver Time are available.
      1. For Gas Saver Flow, enter the amount of carrier gas used in the inlet.
      2. For Gas Saver Time, enter the time after injection when the carrier gas flow drops to the gas saver flow amount.
  • If Split is selected for the Inlet Mode, enter parameters for column 1 and/or column 2 and the front and/or back inlet, depending on the capillary configuration.
    1. For the Split Ratio, enter the ratio of split flow to column flow.
    2. For Split Flow, enter the rate of flow from the split/purge vent.
    3. For Total Flow, enter the total flow rate into the inlet. Total flow rate into the inlet. Total Flow is the sum of the split flow, column flow, and septum purge flow. When the total flow is changed, the split ratio and split flow change while the column flow and pressure remain the same.
    4. For the Gas Saver, select Yes or No for Column 1 Front Inlet Gas Saver and/or Column 2 Back Inlet Gas Saver. Select No, and the flow of carrier gas will not be reduced; select Yes to reduce the flow of carrier gas into the inlet and out of the split vent after the completion of the injection. This does not alter the carrier gas flow through the column. When Yes is selected, Gas Saver Flow and Gas Saver Time are available.
      1. For Gas Saver Flow, enter the amount of carrier gas used in the inlet.
      2. For Gas Saver Time, enter the time after injection when the carrier gas flow drops to the gas saver flow amount.
  • If Pulsed Splitless is selected for the Inlet Mode, enter parameters for column 1 and/or column 2 and the front and/or back inlet, depending on the capillary configuration.
  • For Purge Time, enter the time after the beginning of the run when the purge valve opens.
    1. For Purge Flow, enter the rate of flow from the purge vent.
    2. For Total Flow, enter the actual flow rate to the inlet during a Pre-Run and a run before purge time.
    3. For Pulse Pressure, enter the inlet pressure during pulse duration.
    4. For Pulse Duration, enter the time from run start to when pulse pressure ends.
    5. For the Gas Saver, select Yes or No for Column 1 Front Inlet Gas Saver and/or Column 2 Back Inlet Gas Saver. Select No, and the flow of carrier gas will not be reduced; select Yes to reduce the flow of carrier gas into the inlet and out of the split vent after the completion of the injection. This does not alter the carrier gas flow through the column. When Yes is selected, Gas Saver Flow and Gas Saver Time are available.
      1. For Gas Saver Flow, enter the amount of carrier gas used in the inlet.
      2. For Gas Saver Time, enter the time after injection when the carrier gas flow drops to the gas saver flow amount.
  • If Pulsed Split, is selected for the inlet mode, enter parameters for column 1 and/or column 2 and the front and/or back inlet, depending on the capillary configuration.
    1. For Split Ratio, enter the ratio of split flow to column flow.
    2. For Split Flow, enter the rate of flow from the split/purge vent.
    3. For Total Flow, enter the total flow rate into the inlet. Total flow is the sum of the split flow, column flow, and septum purge flow. When the total flow is changed the split ratio and split flow change while the column flow and pressure remain the same.
    4. For Pulse Pressure, enter the inlet pressure during pulse duration.
    5. For Pulse Duration, enter the time from run start to when pulse pressure ends.
    6. For the Gas Saver, select Yes or No for Column 1 Front Inlet Gas Saver and/or Column 2 Back Inlet Gas Saver. Select No, and the flow of carrier gas will not be reduced; select Yes to reduce the flow of carrier gas into the inlet and out of the split vent after the completion of the injection. This does not alter the carrier gas flow through the column. When Yes is selected, Gas Saver Flow and Gas Saver Time are available.
      1. For Gas Saver Flow, enter the amount of carrier gas used in the inlet.
      2. For Gas Saver Time, enter the time after injection when the carrier gas flow drops to the gas saver flow amount.
  • If Solvent Vent is selected for the Inlet Mode, enter parameters for column 1 and/or column 2 and the front and/or back inlet, depending on the capillary configuration.
    1. For Solvent Vent Time, enter the time after sample injection when the split vent is open to vent solvent.
    2. For Solvent Vent Flow, enter the Flow rate out of the split vent during Solvent Vent Time.
    3. For Solvent Vent Pressure, enter the inlet pressure during Solvent Vent Time.
    4. For the Gas Saver, select Yes or No for Column 1 Front Inlet Gas Saver and/or Column 2 Back Inlet Gas Saver. Select No, and the flow of carrier gas will not be reduced; select Yes to reduce the flow of carrier gas into the inlet and out of the split vent after the completion of the injection. This does not alter the carrier gas flow through the column. When Yes is selected, Gas Saver Flow and Gas Saver Time are available.
      1. For Gas Saver Flow, enter the amount of carrier gas used in the inlet.
      2. For Gas Saver Time, enter the time after injection when the carrier gas flow drops to the gas saver flow amount.

Agilent 7890 Switch Mode

When using the Agilent 7890 GC, if the Inlet Mode is available and not currently set to Split or Pulsed Split, there is an option to set the septum purge mode to Standard or Switched mode. Switched mode shuts off the septum purge flow, increasing the amount on column.
