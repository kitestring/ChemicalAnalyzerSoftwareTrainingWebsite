Mass Spectrometer Methods Overview

Introduction

The Mass Spectrometer (MS) Method is the only method that is always required when running an analysis. It defines the Mass Spectrometer parameter values used for the sample collection, ion source type, and acquisition duration.

It is used extensively during installation to set up and optimize the instrument. It is also used any time you want to verify that the Mass Spectrometer is functioning properly.

Specific details for each MS Method parameter is organized by tab. Click on the desired link for the corresponding MS Method parameter information:
