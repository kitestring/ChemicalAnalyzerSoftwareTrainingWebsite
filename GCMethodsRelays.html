Gas Chromatograph Methods: Relays

  • Enter the Valve/Relay Settings. Use Add or Remove to create or remove a setting. Select On or Off for each valve in the chart.

Generic Configuration

  • Enter the transfer line temperature in °C. Generally choose a temperature value of 10 °C above the maximum oven temperature for the Gas Chromatograph as long as this temperature value is compatible with the column stationary phase. If the temperature displays as –1 because the method was created using a previous version of the software, update the temperature.
