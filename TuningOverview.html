Tuning Overview

Introduction

Regular instrument tuning and the associated system checks is critical for consistent and optimal instrument performance. It is also necessary to monitor for problems such as leaks and tune failures. It is recommended to run a daily instrument check using a Tune Validation. Quality control methods offer the user a tool for automating tuning tasks at specified triggers. Below is the list of each of the tuning tasks and system checks.

  • Auto-Tune
  • Source and Analyzer Focus
  • Mass Calibration
  • Gain Optimization
  • Detector Measurement
  • Leak Check
  • Tune Validation

All of the tuning steps shown above are executed using the Acquisition Queue. It is highly recommended to include the Mass Spectrometer and Gas Chromatographic methods that will be used to acquire data. Many of the parameters within these methods have an impact on the fragmentation behavior and these conditions should be kept constant from tuning to data acquisition. Below is a list of some of the method parameters that will have an effect ion fragmentation:

  • Carrier Gas Type
  • Flow Rate
  • Ion Source Temperature
  • Emission Current
  • Electron Energy
