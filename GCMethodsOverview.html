Gas Chromatograph Methods

Introduction

The following section explains each area of the Chromatograph Method worksheet and how to complete them to create a Chromatograph Method. Depending on the hardware selected, different areas of the worksheet are active. Parameter values entered in one area of the worksheet may activate or inactivate other areas of the worksheet. Most pages of the worksheet now have an indication light (red, yellow, or green) and corresponding text box at the top. The text box will give additional information to explain the indication light. The color of the indication light for each page is also shown in the left navigation pane of the method.

Tool Bar

As described in the Method Building Overview, GC methods have all the same tool bar buttons as do other methods. However, unique to GC methods is the Transfer Method () button. This button will immediately load all the settings defined in the method to the GC, the affected parameters are listed below.

  • Primary GC Oven Temperature
  • Secondary GC Oven Temperature (if applicable)
  • Modulator Temperature (if applicable)
  • Inlet Temperature
  • Transfer Line Temperature
  • Flow Rate

This can be used to automate cooling the front end prior to maintenance, or getting the front to the desired set points prior to acquisition without having to do so manually or via the acquisition queue.

Specific details for each GC Method parameter is organized by tab. Click on the desired link for the corresponding GC Method parameter information:

  • Model Selector
  • Columns
  • Inlet
  • Temperature Zones
  • Auxiliary Zones
  • Relays
  • GCxGC (if applicable)
