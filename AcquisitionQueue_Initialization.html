Acquisition Queue: Initialization

Initialization

After clicking the start button to begin a sequence, ChromaTOF will load the parameters defined in the acquisition methods, most can be monitored in the acquisition queue. The following is a list of parameters that can be monitored during initialization. The video below shows these parameters reaching their set point prior to beginning the acquisition.

  • GC Oven Temperature
  • Modulator Oven Temperature (GCxGC only)
  • Secondary Oven Temperature (GCxGC only)
  • Chiller or LN2 Dewar Status (GCxGC only)
  • Transfer Line Temperature
  • Source Temperature
  • Mass Spectrometer Status
A demonstration of how to monitor the parameters listed above during initialization.

If all the above parameters have reached their starting value and the acquisition has not started, it is likely a parameter not directly monitored by ChromaTOF is the rate limiting factor. An example of this is the inlet temperature.
