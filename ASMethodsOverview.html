Auto Sampler Methods

Manual Start

Manual Start allows the user to prepare the system (for example, infusions) and then determine when the collection should be started. When the system is ready to start the acquisition, the Manual Start dialog box appears. Select Start to begin the acquisition.

For Contact Closure, the system will start collection once a signal is received on the remote interface connection on the back of the instrument.

Agilent 7693 and 7650

When you select the 7693 and 7650 AS Parameters box on the left (highlighted in yellow in the following screenshot) the following parameters appear and are selectable. Edit the following parameters as necessary. Select “Enable Overlapped Operation” and the autosampler will begin preparing the sample before the GC is ready. Select “Configure Advanced Parameters” to edit advanced parameters.

  1. Enter the syringe size; this may update some other volume and speed parameters accordingly.
    • It is recommended to review all parameters after a syringe size is changed to ensure desired behavior.
    • If the Configure Advanced Parameters box is checked, a button will be available to Use Default Speeds.
  2. Set the desired Pre-Injection Wash parameters for Solvent A, Solvent B, and the Sample.
    1. For Repetitions, set the number of times the sample is drawn into the syringe and expelled into the assigned waste bottle.
    2. For Volume, select the desired volume from the drop-down box. Can be set at 10%, 20%, 30%, 40%, and 80%.

    The following parameters are available only if the Configure Advanced Parameters box is checked.

    • Draw Speed—Enter the desired speed of the syringe plunger during sample injection. The Draw Speed will automatically be set to a default speed based on the selected Syringe Size being used.
    • Dispense Speed—Enter the desired plunger speed for the solvent ejection if desired. The Dispense Speed will automatically be set to a default speed based on the selected Syringe Size being used.
    • Viscosity Delay—Enter the length of pause for viscous samples to be drawn into the syringe. Can be set between 0 and 7.
    • Depth Offset—Applicable only for the Sample Parameter. Can be set between -2 and 30. This value sets the depth of the syringe in mm from the bottom of the sample vial; a value of 0 is 2 mm from the bottom. This feature is especially helpful for samples with multiple layers where the sample of interest is not at the normal 2 mm from the bottom of the autosampler vial.
  3. Set the desired Sample Pumps Injection parameters.
    • Repetitions—Set the number of times the solvent is drawn into the syringe and expelled into the assigned waste bottle.

    The following parameters are available only if the Configure Advanced Parameters box is checked.

    • Draw Speed—Enter the desired speed of the syringe plunger during sample injection. The Draw Speed will automatically be set to a default speed based on the selected Syringe Size being used.
    • Dispense Speed—Enter the desired plunger speed for the solvent ejection if desired. The Dispense Speed will automatically be set to a default speed based on the selected Syringe Size being used.
    • Viscosity Delay—Enter the length of pause for viscous samples to be drawn into the syringe. Can be set between 0 and 7.
    • Depth Offset—Applicable only for the Sample Parameter. Can be set between -2 and 30. This value sets the depth of the syringe in mm from the bottom of the sample vial; a value of 0 is 2 mm from the bottom. This feature is especially helpful for samples with multiple layers where the sample of interest is not at the normal 2 mm from the bottom of the autosampler vial.
  4. Set the desired Sample Injection parameters.
    1. For the Volume, enter the sample volume amount.
    2. Set the desired Air Gap volume. This is the amount of air separating the sample from the end of the needle.
    3. For Viscosity Delay, enter the length of pause for viscous samples to be drawn into the syringe. Can be set between 0 and 7.
    4. Set the Pre Dwell between 0 and 1 minutes. This time is how long the syringe needle remains in the inlet prior to injection of the sample. This is commonly used when a hot needle injection is desired in order to reduce discrimination of high boiling compounds and promote volatilization of the liquid sample in the needle.
    5. Set the Post Dwell between 0 and 1 minutes. This is how long the needle remains in the inlet after the injection.
    6. Set the Depth Offset between -2 and 30. This determines the sampling depth.
    7. For the Plunger Speed, select Fast or Slow.

    The following parameters are available only if the Configure Advanced Parameters box is checked.

    • Draw Speed—Enter the desired speed of the syringe plunger during sample injection. The Draw Speed will automatically be set to a default speed based on the selected Syringe Size being used.
    • Dispense Speed—Enter the desired plunger speed for the solvent ejection if desired. The Dispense Speed will automatically be set to a default speed based on the selected Syringe Size being used.
  5. Set the desired Post-Injection Wash parameters for Solvent A and Solvent B.
    1. For Repetitions, set the number of times the sample is drawn into the syringe and expelled into the assigned waste bottle.
    2. For Volume, select the desired volume from the drop-down box. Can be set at 10%, 20%, 30%, 40%, and 80%.

    The following parameters are available only if the Configure Advanced Parameters box is checked.

    • Draw Speed—Enter the desired speed of the syringe plunger during sample injection. The Draw Speed will automatically be set to a default speed based on the selected Syringe Size being used.
    • Dispense Speed—Enter the desired plunger speed for the solvent ejection if desired. The Dispense Speed will automatically be set to a default speed based on the selected Syringe Size being used.
    • Viscosity Delay—Enter the length of pause for viscous samples to be drawn into the syringe. Can be set between 0 and 7.

L-PAL3

L-PAL3 is available only when the corresponding CPK option exists. When the LECO L-PAL3 autosampler is selected, the L-PAL3 box is selectable (highlighted in yellow). Refer to the following screenshot. Choose and customize the desired injection procedure by completing the following instructions:

  1. Click the “Select the injection procedure” drop-down list, and then select a procedure from the menu. The procedure parameters are then loaded into a grid.
    • L-PAL3 Injection Procedure templates are provided for Headspace, Liquid, and SPME. Procedure names denoted with “in tray” will extract the sample from the tray, while those denoted with “using Agitator” move the vial to the agitator for heating and conditioning prior to extraction.
  2. Edit the Value column of the grid to customize the procedure, if desired.
  3. Select the Prep Ahead checkbox to enable overlapping when feasible. The L-PAL3 will prepare the next sample while the prior sample is being collected.
  4. If the method is going to be used to collect additional data, select the Update Configuration button to update the method's internal copy of the L-PAL3 configuration to match the latest synched configuration. The Update Configuration button will be disabled if the method is already using the latest synched configuration.

Rail

When Rail is selected, there are three options that are selectable: Liquid, Head Space, and SPME.

Rail: Liquid

If Liquid is selected, the following Liquid Cycle Parameters appear when the Liquid Cycle Parameters box on the left is selected.

  1. For Syringe, select the drop-down arrow and then select a syringe from the list. A blank cell means the autosampler is not On or is not connected.
  2. For Sample Volume, enter the sample volume amount in microliters (μL).
  3. For Air Volume, enter the volume of air to aspirate after the syringe needle is moved out of the sample liquid.
  4. For Solvent A Pre-Wash, enter the number of times solvent A is drawn into the syringe and expelled into the assigned waste bottle.
  5. For Solvent B Pre-Wash, enter the number of times solvent B is drawn into the syringe and expelled into the assigned waste bottle.
  6. For Sample Pre-Wash, enter the number of times a sample solution is drawn into the syringe and expelled into the assigned waste bottle.
  7. For Viscosity Delay, enter the length of pause for viscous samples to be drawn into the syringe.
  8. For Injection Rate, enter the speed the sample is expelled into the injector.
  9. For Pre-Injection Delay, enter how long the needle is to heat before injection.
  10. For Post-Injection Delay, enter how long the syringe remains in the injector after the plunger has been depressed.
  11. For Solvent A Post–Wash, enter the number of times to rinse the syringe after sample injection. (Minimizes cross contamination with the next sample.)
  12. For Solvent B Post-Wash, enter the number of times to rinse the sample out of the syringe. (Minimizes cross contamination with the next sample.)
  13. For Fill Volume, enter a value to control the filling of the syringe. Air bubbles can remain below the plunger after the first pull up. If the plunger is moved up and down several times, the air bubbles are worked out; therefore, the syringe can be completely filled even when using very small sample volumes.
  14. For Fill Speed, enter a value to specify the speed of plunger movement (used in all syringe filling operations).
  15. For Fill Strokes, enter the number of fill strokes. All fill strokes, except the last one, use the selected fill volume. If the selected sample volume is higher than the fill volume, the sample volume is used for all fill strokes. If zero is selected, the plunger is pulled up only once using the sample volume value.

Rail: Head Space

If Head Space is selected, the following Head Space Cycle Parameters appears when the Head Space Cycle Parameters box on the left is selected. Also, the Agitator parameters box appears.

  1. For Syringe, select the arrow and select a syringe from the list. A blank cell means the autosampler is not on or is not connected.
  2. For Syringe Temperature, enter the temperature of the syringe in ºC.
  3. For Syringe Flush Time, enter the syringe flush time in minutes.
  4. For Sample Volume, enter the sample volume amount in milliliters.
  5. For Pullup Delay, enter a delay time in seconds between sample pullup and ejection while filling the syringe.
  6. For Pre-Injection Delay, enter the time duration in seconds that allows the needle to heat prior to injection.
  7. For Fill Speed, enter the speed (μL/sec) of the plunger movement used in all syringe-filling operations.
  8. For Injection Speed, enter the speed (μL/sec) of plunger movement for sample injection.
  9. For Post-Injection Delay, enter the time duration in seconds that the syringe is to remain in the injector after the plunger has been depressed.
  10. For Agitator, select the Agitator Box on the left. Select the Use Agitator checkbox when a sample will be extracted in the incubator. When the checkbox is checked, the following parameters are available.
    1. For Incubate Temperature, enter a value from 30 °C to 200 °C or zero. Enter zero for sample in tray at ambient temperature without agitation.
    2. For Incubate Time, enter a value from 00:00:00 to 23:59:59. Incubate time allows the sample to be preheated prior to sampling. The default is 5 minutes.
    3. For Agitator Speed, enter a value in rpm from 100 to 750 rpm to set the agitator speed during incubation time.
    4. For Agitator Speed when Extracting (rpm), enter a value in rpm from 100 to 750 rpm to set the agitator speed during extraction of the sample.
    5. For Agitator On Time, enter a value from 0 to 99 seconds. Set to zero to turn Off agitation during pre-incubation and extraction. The default is 5 seconds.
    6. For Agitator Off Time, enter a value from 0 to 99 seconds.

Rail: SPME

If SPME is selected, the following SPME Cycle Parameters appears when the SPME Cycle Parameters box on the left is selected. Also, the Agitator parameters box appears.

  1. For Syringe, select the arrow to select the desired syringe. The entries that display are user defined names entered on the CTC autosampler. A blank cell means the autosampler is not on or is not connected.
  2. For Action On Injector Wait Exceeded, select either Abort or Continue to define the course of action when the SPME needle has waited for the GC to become ready for a longer period of time than was specified for the maximum injector wait.
  3. For Sample Source, select the drop-down arrow to select the source for the sample.
  4. For Action On Over Incubation, select the drop-down arrow and select either Abort or Continue to define the course of action when the sample has incubated longer than the maximum allowed time.
  5. For Derivatizing Agent Extract Time, enter the time for the Derivation Agent extract. If a Derivation Agent will not be used, enter zero.
  6. For Derivatization Agent Vial Penetration, enter the total distance the fiber will extend into the derivatization agent vial. This is the sum of the needle penetration distance and the fiber extension distance.
  7. For Derivatization Agent Vial Penetration-Needle, enter the distance the needle will be injected. This is the amount of the total penetration that the needle extends; the rest is fiber extension. This value must be greater than 10 mm and less than the value entered for the Derivatization Agent Vial Penetration.
  8. For Internal Standard Vial Penetration, enter the total distance the fiber will extend into the internal standard vial. This is the sum of the needle penetration distance and the fiber extension distance.
  9. For Internal Standard Vial Penetration-Needle, enter the distance the needle will be injected. This is the amount of the total penetration that the needle extends; the rest is fiber extension. This value must be greater than 10 mm and less than the value entered for the Internal Standard Vial Penetration.
  10. For Sample Extract Time, enter a value from 00:00:10 to 23:59:59 minutes. This is the amount of time desired for the SPME fiber to be exposed to the sample.
  11. For Sample Vial Penetration, enter the total distance the fiber will extend into the sample vial. This is the sum of the needle penetration distance and the fiber extension distance.
  12. For Sample Vial Penetration – Needle, enter the distance the needle will be injected. This is the amount of the total penetration that the needle extends; the rest is fiber extension. This value must be greater than 10 mm and less than the value entered for the Sample Vial Penetration.
  13. For Injector Penetration, enter the total distance the fiber will extend into the injector (the distance from the top of the injector nut to the end of the fiber). The injector penetration is the sum of the needle penetration distance and the fiber extension distance.
  14. For Injector Penetration-Needle, enter the distance the needle will be injected. This is the amount of the total penetration that the needle extends; the rest is fiber extension. This value must be greater than 10 mm and less than the value entered for the Injector Penetration.
  15. For Desorb Time, enter a value from 00:00:10 to 23:59:59 minutes. Desorb Time indicates the time in injector.
  16. For Fiber Bakeout Time, enter a value from 00:00:00 to 23:59:59 minutes. Fiber Bakeout Time indicates the time for baking out the fiber after desorption in the optional bakeout station.
  17. For Fiber Bakeout Penetration, enter the total distance the fiber will extend for the fiber bakeout. This is the sum of the needle penetration distance and the fiber extension distance.
  18. For Fiber Bakeout Penetration-Needle, enter the distance the needle will be injected. This is the amount of the total penetration that the needle extends; the rest is fiber extension. This value must be greater than zero mm and less than the value entered for the Fiber Bakeout Penetration.
  19. For Maximum Injector Wait, enter the maximum length of time for the SPME needle to wait for the GC to be ready. The default is 30 seconds.
  20. For Maximum Over Incubation, enter a value to define the maximum length of time for a sample to incubate in the agitator. The default is 10 seconds.
  21. For Agitator, select the Agitator Box on the left. Select the Use Agitator checkbox when a sample will be extracted in the incubator. When the checkbox is checked, the following parameters are available.
    1. For Incubate Temperature, enter a value from 30 °C to 200 °C or zero. Enter zero for sample in tray at ambient temperature without agitation.
    2. For Incubate Time, enter a value from 00:00:00 to 23:59:59. Incubate time allows the sample to be preheated prior to sampling. The default is 5 minutes.
    3. For Agitator Speed, enter a value in rpm from 100 to 750 rpm to set the agitator speed during incubation time.
    4. For Agitator Speed when Extracting (rpm), enter a value in rpm from 100 to 750 rpm to set the agitator speed during extraction of the sample.
    5. For Agitator On Time, enter a value from 0 to 99 seconds. Set to zero to turn Off agitation during pre-incubation and extraction. The default is 5 seconds.
    6. For Agitator Off Time, enter a value from 0 to 99 seconds.
