  • Auto Select

    The auto select features automatically selects a method linked to a sample when enabled. It functions differently when used from within in Acquired Samples versus the Acquisition Queue. When utilized from within Acquired Samples, an uneditable version of the method used to acquire the currently viewed sample will automatically be selected. This allows the user to maintain a record of all the acquisition methods for each acquired sample. These records do not change even if the current version of the method is altered. To open an editable version of the method click the open method button and select it.

    Auto Select behavior when within Acquired Samples

    If a method is visible when viewing the Acquisition Queue and Auto Select is enabled, an editable version of the method selected in the corresponding row in the acquisition queue will automatically be shown. This will automatically updates as different rows in the acquisition queue are selected.
