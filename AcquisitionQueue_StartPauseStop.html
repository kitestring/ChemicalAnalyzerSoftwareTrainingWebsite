Acquisition Queue: Start, Pause or Stop the Queue

Start a Sequence

To start a sequence click the start button found in the acquisition queue tool bar.

Start a sequence using the start button .

Common Start Error

When the start button is clicked ChromaTOF will verify that each of the acquisition methods used in the sequence are valid. If a queued method fails this check the status in the corresponding row will prompt “Error” and the sequence will not begin. Check the system log for a description of the error. The video below presents an example of this; one of the GC methods is not using the Global GC Settings.

Pre-acquisition method validation failure preventing an sequence from starting.

Pause the Acquisition

If the pause button is clicked during an acquisition, the current acquisition will be unaffected and continue to run. However, the following injection will not automatically begin if the acquisition queue is left in a paused state. An example of this is shown in the video below. To continue acquiring toggle the start button.

A depiction of a sequence when in a paused state.

Stop Current Acquisition

To abort an actively acquiring sequence click the “Stop Current Acquisition” button as demonstrated in the following video. Note, the mass spectrometer will abort however, the GC will complete the loaded method. To stop the GC use direct GC interface.
