Acquisition Queue: Add and Remove Rows

Add Rows

Rows can be individually added by clicking the insert button which can be found in the in the acquisition queue tool bar or the ribbon. When adding a row to an empty queue the column values from the most recently acquired sample will automatically be populated into the newly added row. When adding subsequent rows the column values from the row directly above will be copied into the new row. Note, the vial number will auto-increment by 1 for subsequent rows added to the acquisition queue. The video shown below provides a demonstration of the row adding behavior described.
