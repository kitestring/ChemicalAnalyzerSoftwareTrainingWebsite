Leak Check

Introduction

A leak check examines the data coming off the instrument for indications that there is a leak somewhere in the system. A leak check can be run manually via the acquisition queue, see the video below, or set to run at regular intervals using the Quality Control Method.

Adding a Leak to the Acquisition Queue

Viewing Leak Check Results

The results of a leak check can be viewed in the log or in the information section of a Performance Report. A leak check will only show if there is a passing or failing result, it does not provide any detail on the relative abundances of the ions measured for the leak check. For more detailed information on the relative abundances of the atmospheric ions, 28 & 32, see the summary section of the Performance Report. It is worth nothing that leak results listed in in the summary section of the Performance Report are extracted from the Tune Validation and not the Leak check acquisition. For more information on a Performance Report see the Tune Validation section of tuning.
